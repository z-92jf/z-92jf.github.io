# 下载我们的应用

> 下载SERVICES FOR Windows

## 下载前的准备

1.你需要准备一个计算机设备，该设备操作系统应为Windows 11（经测试，在该版本上有少量bug）或Windows 10（未在该版本进行测试）

2.你需要准备至少1000Mib的储存空间

3.你需要准备至少8GB的运行内存

4.你需要管理员权限^①^

①：也可以选择“仅为自己安装“

## 准备安装文件

进入[我们的Github仓库](https://github.com/z-92jf/SERVICES FOR Windows)直接下载我们的所有安装包。截至2026年4月13日，建议安装的版本为V2.0.0，此版本推荐的原因为bug较少。

## 开始安装

双击下载好的安装包（可能需要提前解压），按照流程安装。

## 答疑解惑

Q：为什么我无法安装？

A：尝试右击安装包，选择”以管理员模式运行“或”以管理员身份运行“。
